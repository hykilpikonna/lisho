# Vue 3 TS + Vite + Vue Router + SASS Template

Demo: https://template.hydev.org/

Usage:

1. Clone this repo
2. Change project name in package.json
3. `yarn install`
4. `yarn serve`

## Info

Web Framework: `Vue 3 + Typescript + Vite + Vue Router + SASS + Element Plus + Class Components`
